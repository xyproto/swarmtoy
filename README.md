SwarmToy
========

* A frenetic game.
* An open ended sandbox.

Requirements
------------

* python2
* pygame
* psyco (optional)

Input
----

* `f` for toggling fullscreen
* `ESC` or `q` to quit
* `Space` to clear
* `p` to pause
* left mousebutton to place food
* center mousebutton to place snack
* right mousebutton to place a creature
